# Kota Chinmaya Sree Portfolio

## Overview

This repository contains the personal portfolio website for Kota Chinmaya Sree, an Industrial Biotechnology undergraduate focusing on biotechnology, healthcare, and computational research.

The portfolio is built as a static web page using:
- `index.html` for page structure and content
- `style.css` for visual styling, glassmorphism, animations, and responsive layout
- `script.js` for interactive behavior, animations, canvas effects, modals, and form handling

## Key Sections

- **Home**: Introduction with animated hero text, resume download link, stats, and social links
- **About**: Professional summary, education timeline, and personal details
- **Skills**: Biotechnology, lab, computational, research, soft skills, and language proficiencies
- **Projects**: Featured research and academic projects with modal details
- **Certifications**: Credential cards with links to certificates
- **Contact**: Contact information and a contact form

## Highlights

- Uses GSAP and ScrollTrigger for animations
- Includes a custom particle canvas and mouse glow effect
- VanillaTilt for interactive 3D tilt cards
- Responsive navigation and modal interactions
- Strong emphasis on biotech, computational biology, and research experience

## Local Preview

To preview the portfolio locally:
1. Open `index.html` in a web browser

Alternatively, if you want a local web server, use any static server, for example:

```bash
# Python 3
python -m http.server 8000
```

Then visit:

```
http://localhost:8000
```
